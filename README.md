# 基于STM32的智能考勤系统

## 项目概述

本项目是一个基于STM32微控制器的智能考勤系统，集成了RFID卡识别、指纹识别、WiFi通信等多种技术，实现了高效、准确的人员考勤管理功能。系统具备本地存储、网络同步、时间管理等功能，适用于企业、学校等场所的考勤管理需求。

## 功能特性

- **多种识别方式**：支持RFID卡识别和指纹识别双重验证
- **本地存储**：支持用户信息和考勤记录本地存储
- **网络同步**：通过ESP8266 WiFi模块实现考勤数据云端同步
- **时间管理**：自动时间同步、跨天处理、考勤规则判断
- **用户管理**：支持用户添加、删除、信息查询等管理功能
- **OLED显示**：直观的图形化用户界面
- **音频提示**：操作音效和状态提示音

## 硬件组成

- **主控芯片**：STM32F103RCT6
- **显示屏**：0.96寸OLED显示屏 (SSD1306)
- **RFID模块**：RC522射频识别模块
- **指纹模块**：ZW101光学指纹识别模块
- **WiFi模块**：ESP8266
- **音频模块**：DAC音频输出
- **按键输入**：上下键、OK键、ESC键

## 软件架构

### 主要模块

1. **应用层 (application/)**
   - `runtime_manager.c` - 运行时状态管理
   - `attendance_app.c` - 考勤应用逻辑
   - `key_input.c` - 按键输入处理
   - `oled_ssd1306.c` - OLED显示驱动
   - `rc522_app.c` - RC522 RFID模块驱动
   - `zw101_app.c` - ZW101指纹模块驱动
   - `storage_manager.c` - 数据存储管理

2. **通信协议 (commen/)**
   - `Com_protocol.c` - STM32与ESP8266通信协议
   - `Com_debug.c` - 调试输出工具

3. **驱动层 (driver/)**
   - 各种硬件驱动模块

4. **后端服务器 (backend-server/)**
   - Node.js + Express + MongoDB
   - MQTT协议处理考勤数据

5. **前端应用 (frontend/)**
   - 微信小程序客户端

## 技术特点

### 1. 实时操作系统
- 使用FreeRTOS实现多任务并发处理
- 合理的任务调度和资源管理

### 2. 通信协议
- 自定义串口通信协议，支持帧头、数据长度、CRC校验、帧尾
- 帧尾从0x55改为0xCC，避免与数据内容冲突
- 支持心跳包、时间同步、数据上传等多种消息类型

### 3. 数据管理
- 用户信息本地存储
- 考勤记录自动归档
- 支持数据备份和恢复

### 4. 用户体验
- 直观的OLED界面显示
- 按键操作逻辑清晰
- 音频反馈及时

## 项目部署

### 硬件连接

STM32F103RCT6 <---> RC522 (SPI) STM32F103RCT6 <---> ZW101 (UART) STM32F103RCT6 <---> ESP8266 (UART) STM32F103RCT6 <---> SSD1306 (I2C) STM32F103RCT6 <---> 按键矩阵 (GPIO) STM32F103RCT6 <---> DAC音频 (DAC)


### 软件编译
1. 使用STM32CubeIDE或Keil MDK编译STM32固件
2. 使用Arduino IDE编译ESP8266固件
3. 部署Node.js后端服务
4. 配置微信小程序前端

### 系统配置
- 配置WiFi连接参数
- 设置MQTT服务器信息
- 校准时间同步机制

## 使用说明

### 正常打卡
1. 用户刷卡或按指纹
2. 系统验证身份
3. 根据时间判断考勤状态（准时、迟到、早退等）
4. 显示打卡结果并上传数据

### 用户录入
1. 短按UP键进入录入模式
2. 刷卡绑定员工信息
3. 录入指纹模板
4. 保存用户信息

### 系统管理
- 长按ESC键进入用户列表
- 上下键浏览用户
- 短按OK键删除用户
- 长按OK键退出用户列表

### 系统功能
- 短按UP键（待机模式）：时间同步
- 短按DOWN键（待机模式）：重启ESP8266
- 短按OK键（录入模式）：退出录入模式
